Introduction

The language application grid (Lapps) project requires development software support. Here are the software, and the required ones are in bold font style.
  • Development Language : Lapps chooses Java as 1st development language, according its acceptable running speed, development easiness and a large number of library. E.g. OpenNLP, Stanford NLP, and UIMA are all Java projects.
  • Dependency Management : Maven is the most popular one for Java development.
  • Source Code Management : With several developers in different groups, we would like to hold our source codes in Github. It will be much easier to cooperate.
  • Development Environment : Eclipse or IntelliJ are IDE of Java development. They are optional and helpful for Java syntax check and prediction.

What is LappsVM?

We design Lapps software version management tool named LappsVM, which is inspired by GVM project. LappsVM is much simple. Here is the steps to install maven through LappsVM.

[shicq@chilote ~]$ curl -s eldrad.cs-i.brandeis.edu:8080/lappsvm/ | bash ↵ Install LappsVM Thanks for using Lapps Version Manager! . . source "/home/j/llc/shicq/.lappsvm/bin/lappsvm-init.sh" . . Enjoy!!! [shicq@chilote ~]$ source "/home/j/llc/shicq/.lappsvm/bin/lappsvm-init.sh" ↵ Load LappsVM commands [shicq@chilote ~]$ lappsvm help ↵ Test LappsVM commands [shicq@chilote ~]$ lappsvm install maven ↵ Install Maven through LappsVM downloading: maven 3.2.1 [shicq@chilote ~]$ mvn -version ↵ Test installed Maven
